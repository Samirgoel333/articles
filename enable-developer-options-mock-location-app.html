Android setup guide

Enable Developer options and select Mock Loc

Before Mock Loc can fake your GPS, Android needs to know which app is allowed to provide mock location data. This guide walks you through enabling Developer options and choosing Mock Loc — a normal Android permission flow that does not require root access.

Why Android requires this setting

In short: Android only allows one app at a time to supply mock GPS coordinates. Selecting Mock Loc tells the system you trust it for location testing — other apps cannot quietly take over this permission.

Mock location is a developer feature built into Android. It exists so testers, QA engineers, and app developers can simulate being somewhere else without physically travelling there. Mock Loc uses this official setting — you are not hacking the phone or installing risky system tools.

Before you start

Make sure Mock Loc is installed from the Play Store (or your trusted source) before you complete Step 3. Android can only list apps that are already on your device.

You will need

  • An Android phone running a recent version of Android (most phones from the last several years work).
  • Your screen lock PIN, password, or fingerprint — Android may ask for this when enabling Developer options.
  • About 2–3 minutes to complete all three steps.

The menu names below can vary slightly between Samsung, Pixel, OnePlus, Xiaomi, Vivo, Oppo, and other brands. The overall flow is the same on nearly every Android phone.

Step-by-step setup

Follow these three steps in order. If you get stuck, check the device tips or troubleshooting sections below.

1
Open About phone

Open Settings on your phone. Scroll down and tap About phone. On some devices this lives inside System or Software information — look for a screen that shows your Android version and device model.

2
Enable Developer options

Find the row labelled Build number and tap it seven times in quick succession. Android will count down (for example, “You are now 3 steps away from being a developer”). When finished, you will see a message like You are now a developer!

3
Select Mock Loc as the mock location app

Go back to the main Settings screen. Open Developer options (sometimes under System or Additional settings). Scroll to the debugging section and tap Select mock location app (on older Android versions it may say Mock location app). Choose Mock Loc from the list.

What each step does

  • Step 1 — Takes you to the hidden developer unlock screen.
  • Step 2 — Unlocks Developer options in Settings (this is intentional — Android hides advanced tools from casual users).
  • Step 3 — Registers Mock Loc as the one app allowed to send mock GPS data to the system.

Tips by phone brand

If you cannot find a menu item, try these common locations. The setting always exists — it is just labelled or nested differently.

Brand Where to look
Samsung Galaxy Settings → About phone → Software information → Build number. Developer options appear under Settings → Developer options.
Google Pixel Settings → About phone → Build number at the bottom. Developer options under System → Developer options.
OnePlus / Oppo / Realme Settings → About device → Version → tap Build number. Developer options may be under Additional settings.
Xiaomi / Redmi / POCO Settings → About phone → MIUI version (tap 7 times on some models) or Build number. Enable Developer options, then find mock location under Debugging.
Vivo / iQOO Settings → About phone → Software version or Build number. Developer options often under System management.

On Android 12 and newer, the setting is usually called Select mock location app. On older versions it may appear as Allow mock locations with a simple on/off toggle — turn it on, then pick Mock Loc if prompted.

Quick final checklist

Before you open Mock Loc and start mocking, confirm these three items:

Ready to mock?

  • Developer options are enabled on your Android phone.
  • Mock Loc is selected under Select mock location app.
  • You have granted location permission when Mock Loc asks (required for the app to work).

Once complete, open Mock Loc, pick a place on the map, and press Start or Mock. Your device will then report the selected GPS coordinates to other apps.

Troubleshooting

Mock Loc does not appear in the list

Install or update Mock Loc from the Play Store, then reopen Select mock location app. If it still does not show, restart your phone and try again.

Developer options disappeared

Some phones hide Developer options after a restart until you re-enable them. Repeat Step 2 (tap Build number seven times) if the menu is missing.

Mocking does not work in other apps

Confirm Mock Loc is still selected as the mock location app. Also check that Mock Loc's location permission is set to Allow all the time or While using the app as required, and that you have actually started a mock session inside Mock Loc (the app must be running a stationary point, playlist, or itinerary for coordinates to change).

I want to stop using mock locations

Open Developer options → Select mock location app → choose None or No app. Your phone will return to using real GPS from satellites and cell towers.
